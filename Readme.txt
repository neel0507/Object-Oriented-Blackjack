The application is developed using Microsoft Visual Studio Community Edition 2015.

C++11 version should be used to run the program.

To compile using g++ in the Windows, use the following command: "g++ Main.cpp Cards.cpp Log.cpp Player.cpp Computer.cpp -o Pontoon -std=c++11"

It is recommended to use Visual Studio, as Visual Studio is used to develop the application.

The application can also be used by simply running the execution file.
